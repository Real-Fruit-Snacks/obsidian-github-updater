Capabilities

Escape the community store.

Manage beta and unlisted plugins directly from GitHub. Automatic updates, release notes, and smart fallback downloads included.

Install from GitHub

Easily install any plugin not found in the community store by simply entering the username/repo.

Auto-Detect Plugins

A built-in scanner flags unofficial plugins already in your vault and accurately guesses their repo to track them automatically.

Smart Update Checker

Keep your tracked plugins up to date. The plugin fetches the latest releases, renders markdown changelogs, and updates with one click.

Intelligent Fallback

If the developer didn't attach release assets (main.js, manifest.json), the updater automatically falls back to raw source code.

Workflow

Install. Track. Update.

1

Scan Your Vault

Go to settings and hit Scan Vault. The plugin detects any unlisted plugins you've installed manually and tracks them.

2

Check For Updates

The updater checks the GitHub releases page for every tracked repo and lets you read perfectly-rendered markdown release notes.

3

One-Click Install

Click Update. The plugin downloads the assets, replaces your existing plugin files, and reloads it instantly without restarting Obsidian.

Get Started

Install in under a minute.

Recommended

BRAT

  1. Install the BRAT community plugin.
  2. Open BRAT settings and click Add Beta plugin.
  3. Paste this repository URL.
  4. Enable GitHub Updater in your plugin list.

Manual

  1. Download main.js, manifest.json, and styles.css from the latest release.
  2. Create .GitHub Updater/plugins/{{REPO_NAME_LOWER}}/ inside your vault.
  3. Place the three files inside that folder.
  4. Reload GitHub Updater and enable the plugin.
